Calling the microservice: ran locally on port 5001, so called like this with a Http get request:
http://(IP ADDRESS):5001/get-restaurant?time=(value 1-24)

Receving data in a JSON file with the format like this:
[
  {
    "name": "Burger Bomb",
    "phone": "5553234",
    "type": "Fast Food",
    "openHour": 10,
    "closeHour": 21
  }
]
